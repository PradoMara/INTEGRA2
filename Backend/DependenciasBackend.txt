Dependencias Backend

Base de Datos y ORM

@prisma/client: ORM para interactuar con la base de datos PostgreSQL
prisma: CLI y herramientas de desarrollo para Prisma


Framework y Core
express: Framework web para Node.js
cors: Middleware para habilitar CORS
dotenv: Manejo de variables de entorno


Seguridad
jsonwebtoken: Generación y verificación de JWT
bcrypt: Encriptación de contraseñas
helmet: Seguridad HTTP headers
express-rate-limit: Limitador de peticiones


Validación y Sanitización
express-validator: Validación de datos en requests
sanitize-html: Sanitización de contenido HTML
joi: Validación de esquemas


Utilidades
nodemailer: Envío de emails
multer: Manejo de subida de archivos
winston: Logging
morgan: HTTP request logger
date-fns: Manejo de fechas


Testing
jest: Framework de testing
supertest: Testing de HTTP endpoints
vitest: Framework de testing moderno


Desarrollo
typescript: Soporte de TypeScript
ts-node: Ejecución de TypeScript
nodemon: Auto-recarga en desarrollo
@types/node: Tipos de Node.js
@types/express: Tipos de Express


Websockets
socket.io: Comunicación en tiempo real
@socket.io/client: Cliente de Socket.IO
